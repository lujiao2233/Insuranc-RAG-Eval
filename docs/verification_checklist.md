#### 文档分析接口
- **URL**: POST http://localhost:8000/api/v1/analysis/analyze/{document_id}
- **预期结果**: 返回分析任务启动信息

#### 分析状态查询
- **URL**: GET http://localhost:8000/api/v1/analysis/status/{document_id}

### 5. 功能流程验证

#### 上传流程
1. 访问文档上传页面
2. 选择一个PDF/DOCX/XLSX文件
3. 点击上传按钮
4. **预期**: 文件上传成功，提示"上传成功"，不进行自动分析

#### 列表查看流程
1. 访问文档列表页面
2. 查看刚上传的文档
3. **预期**: 文档显示在列表中，分析状态为"未分析"

#### 分析流程
1. 在文档列表中选择未分析的文档
2. 点击"分析"按钮
3. **预期**: 提示"分析任务已启动"，文档状态变为"处理中"
4. 等待几秒后刷新列表
5. **预期**: 文档分析状态变为"已分析"

#### 批量分析流程
1. 上传多个文档
2. 在列表中勾选多个未分析的文档
3. 点击"分析选中文档"按钮
4. **预期**: 提示"已启动N个文档的分析任务"

#### 查看分析结果
1. 点击已分析文档的"查看"按钮
2. **预期**: 显示文档详情对话框，包含元数据和大纲信息
